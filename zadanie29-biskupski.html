Zadanie 29 – Odpowiedzi na pytania

1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

W JavaScript właściwości obiektu można określić na dwa sposoby:
Notacją kropkową: obiekt.nazwa = "wartość";
Notacją tablicową: obiekt["nazwa"] = "wartość";
Przykład:
var osoba = {};
osoba.imie = "Anna";
osoba["nazwisko"] = "Nowak";

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do odwoływania się do pól (właściwości) i metod używamy również notacji kropkowej lub tablicowej. Na przykład:
osoba.imie lub osoba["imie"] – odwołanie do pola;
osoba.przedstawSie() – wywołanie metody.
Metody definiujemy jako funkcje przypisane do obiektu, a ich wywołanie odbywa się przez dodanie nawiasów.

3. Wymień cechy charakterystyczne konstruktora.

Konstruktor w JavaScript to specjalna funkcja służąca do tworzenia nowych obiektów.
Cechy konstruktora:

  • Inicjalizuje nowe właściwości obiektu.
  • Może przyjmować parametry, co pozwala na dynamiczne przypisanie wartości.
  • Wewnątrz konstruktora słowo this odnosi się do nowo tworzonych obiektów.
  • Automatycznie zwraca nowy obiekt, gdy użyjemy słowa kluczowego new.

4. Konstruowanie obiektu na podstawie przykładu – obiekt ma trzy pola oraz jedną metodę.

Przykładowy konstruktor:
function Osoba(imie, nazwisko, wiek) {
   this.imie = imie;
   this.nazwisko = nazwisko;
   this.wiek = wiek;
   this.przedstawSie = function() {
      return "Nazywam się " + this.imie + " " + this.nazwisko + ", mam " + this.wiek + " lat.";
   }
}
var osoba1 = new Osoba("Jan", "Kowalski", 30);

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo this odnosi się do bieżącego obiektu, w kontekście którego jest używane – najczęściej w konstruktorach lub metodach. Dzięki niemu możemy odwoływać się do pól oraz metod danego obiektu.
Oddzielenie pól i metod polega na tym, że właściwości (pola) przypisujemy do this (np. this.imie = "Anna";), a metody definiujemy jako funkcje przypisane do this lub dodawane później do prototypu.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

Przykład konstruktora:
function Samochod(marka, model, rok) {
   this.marka = marka;
   this.model = model;
   this.rok = rok;
   this.wyswietlInfo = function() {
      return this.marka + " " + this.model + ", rok produkcji: " + this.rok;
   }
}
var auto = new Samochod("Toyota", "Corolla", 2020);
Opis: Funkcja Samochod jest konstruktorem, który tworzy obiekt o trzech polach oraz jednej metodzie. Przy wywołaniu z użyciem słowa kluczowego new tworzony jest nowy obiekt, a this wskazuje na ten nowo utworzony obiekt.

7. Znaczenie słowa kluczowego new wraz z przykładem.

Słowo kluczowe new służy do tworzenia nowej instancji obiektu na podstawie konstruktora. Gdy używamy new, JavaScript:

  • Tworzy nowy obiekt.
  • Przypisuje this do nowego obiektu w obrębie konstruktora.
  • Automatycznie zwraca nowo utworzony obiekt.
Przykład:
var osoba = new Osoba("Anna", "Nowak", 25);

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Słowo prototype umożliwia dodawanie właściwości i metod do wszystkich instancji danego konstruktora. Dzięki temu metody te są współdzielone przez wszystkie obiekty, co oszczędza pamięć, gdyż nie są kopiowane do każdego nowego obiektu.
Przykład:
function Student(imie, nazwisko) {
   this.imie = imie;
   this.nazwisko = nazwisko;
}
Student.prototype.przedstawSie = function() {
   return "Cześć, jestem " + this.imie + " " + this.nazwisko;
};
var student1 = new Student("Piotr", "Zieliński");
Opis: Metoda dodana przez prototype jest współdzielona przez wszystkie instancje konstruktora Student. Dzięki temu nie definiujemy tej metody wewnątrz konstruktora, co pozwala na oszczędniejsze zarządzanie pamięcią i spójność zachowania obiektów.
